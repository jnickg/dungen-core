About the App

DunGen is a web application supporting:

  • Customized dungeon generating routines, using a broad tool box of parameterized algorithms ("MazeCarver + DeadEndFiller + RoomCarver = MyNewDungeonMakerRoutine"). This includes:
    • Dungeon layout generating (as described above)
    • Dungeon infestation with traps, enemies, and treasure!
  • Paint yor dungeon's pattern, using a painting palette comrised of all your custom routines, along with community-made routines, then generate it! Voila: randomized dungeons followign the same handmade pattern.
  • Infest your dungeon automatically, or manually, using a number of canonical and community-made Libraries.
  • Interactive full-control editing of your dungeon, every step of the way
  • Save your dungeon, and export to a slew of formats such as PDF, HTML, and so on!

Designed to be both easy-to-use, and highly customizable, and portable to any game system (both tabletop and desktop), it's the last dungeon generator you'll ever need.
